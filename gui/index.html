    Sin notificaciones

    No se encontró FFmpeg

    ViriaRevive necesita ffmpeg en el PATH para descargar, recortar y codificar vídeos.

    1. Opción rápida (Windows, recomendada): winget install "FFmpeg (Essentials Build)" (suele bastar para ViriaRevive: libass, x264…). Alternativa más completa: winget install ffmpeg (release full). Cierra y vuelve a abrir la app.
    2. Sin tocar el PATH: descarga ffmpeg-release-essentials.zip desde gyan.dev/ffmpeg/builds, descomprime y coloca ffmpeg.exe en …/ViriaRevive-main/ffmpeg/bin/ (junto a config.py).
    3. Full solo si lo necesitas: más códecs y librerías (p. ej. algunos filtros extra); para clips y subtítulos suele sobrar con Essentials.

    Si ya lo instalaste, pulsa «Comprobar de nuevo» tras cerrar y abrir la terminal o la app.

    Configuración API de YouTube

    1. Entra en console.cloud.google.com
    2. Crea un proyecto nuevo
    3. Activa YouTube Data API v3
    4. Crea credenciales OAuth 2.0 (aplicación de escritorio)
    5. Descarga el archivo JSON
    6. Guárdalo como client_secrets.json en la carpeta de ViriaRevive
